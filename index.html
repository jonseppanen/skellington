Skellington

Skellington is a barebones minimalistic pseudo-framework targetted at real developers. It uses the minimum amount of crap to get you going.

FAQ
  • What grid does Skellington use?
    Grid? Make your own damn grid. Skellington uses ungrid by default though.
  • Ok, how do I make a button?
    You code it in the CSS file. Like a real developer.
  • Well fuck you.
    Fuck you too.

Skellington's main purpose is two-fold, Firstly, to perform a global blank and reset on all properties to all elements, then make them consistent.

Secondly, set up a screen size based font scaler so everything is readable on all screens. Sizing your text and elements by REM will allow you to take advantage of this.
